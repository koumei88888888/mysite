セットアップ手順（初回のみ）
  1. Google Cloud Console でプロジェクトを作成
  2. 「APIとサービス」→「ライブラリ」→ YouTube Data API v3 を有効化
  3. 「APIとサービス」→「認証情報」→「認証情報を作成」→「OAuthクライアントID」
  4. アプリの種類: ウェブアプリケーション、承認済みリダイレクトURIに https://koumei88888888.github.io/mysite/apps/ym-playlist/ を追加
  5. クライアントIDをコピーして下の欄に貼り付け
  6. 「OAuth同意画面」でテストユーザーに自分のGoogleアカウントを追加

全曲プレイリストジェネレーター

インスト・off vocal・重複を自動除外して Spotify に保存

使い方 / セットアップ手順
  1. Spotify Dashboard でアプリを作成（Web API にチェック）
  2. 「Edit Settings」→ Redirect URIs に https://koumei88888888.github.io/mysite/apps/spotify-playlist/ を追加して保存
  3. Client ID をコピーして下の欄に貼り付け
  4. 「Spotify で認証」→ 認証後このページに自動で戻ってきます
  5. アーティストを検索 → 候補から選択 → プレイリスト作成
Step 1 — Spotify 認証
Redirect URI は https://koumei88888888.github.io/mysite/apps/spotify-playlist/ を使用します。
Spotify Dashboard の Edit Settings → Redirect URIs に上記を登録してください。
